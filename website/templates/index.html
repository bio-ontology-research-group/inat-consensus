Federated Query Example: Proteins by Function

This query demonstrates how to select a taxon from the local Knowledge Graph (e.g., mapped to NCBI) and dynamically query the remote UniProt SPARQL endpoint to find proteins with a specific function.

PREFIX up: 
PREFIX sio: 
PREFIX rdfs: 
PREFIX ex: 

SELECT ?taxon ?protein ?name ?function
WHERE {
  # 1. Local: Find a taxon with an NCBI ID
  ?taxon rdfs:subClassOf+ sio:Taxon .
  FILTER(REGEX(STR(?taxon), "NCBITaxon_"))
  BIND(STRAFTER(STR(?taxon), "NCBITaxon_") AS ?ncbi_id)

  # 2. Remote: Ask UniProt for proteins of this taxon
  SERVICE  {
    BIND(IRI(CONCAT("http://purl.uniprot.org/taxonomy/", ?ncbi_id)) AS ?up_taxon)
    ?protein up:organism ?up_taxon ;
             up:recommendedName/up:fullName ?name ;
             up:annotation ?anno .
    ?anno a up:Function_Annotation ;
          rdfs:comment ?function .
  }
}
LIMIT 5
